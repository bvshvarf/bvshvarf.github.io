User Stories

As a Dataseq Guest:
  • Submit a FastQ file for preprocessing with RNA, WGBS, or ATAC sequencing.
  • Select the specific preprocessing pipeline via radio buttons.
  • Option to sign up for an account displayed on the homepage.
  • Download finished preprocessed data easily.
As a Dataseq User:
  • Save configurations to settings so I don't have to set them again.
  • Personalize the dashboard view on the web application.
  • Understand visuals to better navigate the web application.
  • Provide critique and suggestions for design and experience.
  • Get help with questions or difficulties.
As a Dataseq Administrator:
  • Monitor security to ensure data protection.
  • Receive user activity logs for tracking actions.
  • Limit access control based on the user's role.
  • Easily manage user quotas.
  • Set default preprocessing parameters to reduce repeated work.
  • Receive a detailed report on preprocessing errors.
